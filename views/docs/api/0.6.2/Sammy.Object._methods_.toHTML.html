Sammy.Object toHTML ( )

Comments | Show in GitHub

Renders a simple HTML version of this Objects attributes. Does not render functions. For example. Given this Sammy.Object:

var s = new Sammy.Object({first_name: 'Sammy', last_name: 'Davis Jr.'}); s.toHTML() //=> 'first_name Sammy
last_name Davis Jr.
'
